# 🚢 諾亞方舟 (Noah's Ark)

這是一款互動角色扮演遊戲。

## 🌟 開發緣起
本遊戲原為夏令營創作的角色扮演遊戲，為了能更長久地保存並與更多人分享，特地將其移植至網頁平台。這不僅是一次技術的轉移，更是創意與數位科技結合的實踐。

## 🎨 遊戲特色
* **AI 深度協作**：全方位利用 AI 技術協助開發，涵蓋圖像生成、劇情腳本撰寫及核心程式邏輯調校。
* **豐富原創素材**：內含精心製作的視覺圖像與音效，打造獨特的遊戲氛圍。
* **深度互動系統**：具備完整的任務鏈與劇情系統，包含具挑戰性的「傑哥遊戲」與充滿創意的「木匠任務」。

## 👨‍💻 技術堆疊
* **語言**：HTML5, CSS3, JavaScript (Canvas API)
* **協作工具**：Generative AI (圖像、代碼優化)

## 🙏 致謝
本專案由 **大威老師 (Da-Wei Teacher)** 與 **AI** 歷經 **90 小時** 的密集協作開發完成。感謝這段開發旅程中的每一顆靈感火花。
